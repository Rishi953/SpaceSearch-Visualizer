# Maze visualizer
A program to visualize 4 Algorithms:-
  i. BFS
  ii. DFS
  iii. A*
  iv. Hill Climbing

## Controls
Key                 Action
L-Click             Place Start (Green), End (Red), or Walls (Black)
R-Click             Erase Walls/Nodes 
Space               Start Game from Menu  
Esc                 Return to Menu  
M                   Generate Random Maze
C                   Clear Grid  
B                   Run BFS 
D                   Run DFS
A                   Run A*
H                   Run Hill Climbing

# How to run
1.Install dependencies: pip install -r requirements.txt
2.Run: python main.py
