# Maze visualizer
This is a Mini Project where we try to visualize 4 Algorithms.<br>
1.BFS<br>
2.DFS<br>
3.A*<br>
4.Hill Climbing<br>

## Controls
| Key | Action |
| :--- | :--- |
| **L-Click** | Place Start (Green), End (Red), or Walls (Black) |
| **R-Click** | Erase Walls/Nodes |
| **Space** | Start Game from Menu |
| **Esc** | Return to Menu |
| **M** | Generate Random Maze |
| **C** | Clear Grid |
| **B** | Run BFS |
| **D** | Run DFS |
| **A** | Run A* |
| **H** | Run Hill Climbing |

# How to run
1.Install dependencies: pip install -r requirements.txt<br>
2.Run: python main.py
